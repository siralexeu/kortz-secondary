# The Kortz Center — Secondary Targets Tracker

A small tool for GTA Online players prepping for [The Kortz Center Heist](https://www.rockstargames.com/newswire/article/2525o93834o413/the-kortz-center-heist-now-available-in-gta-online). It's a spreadsheet of all 65 secondary targets — paintings, gemstones, statues, jewelry, crates, and more — so you can see best seconadry target and how much it pays out.

The preview images also help you identify buyer requests, so you know exactly which item you're looking for.

**Status:** active — images being added as in-game captures become available.
