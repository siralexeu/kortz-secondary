# The Kortz Center — Secondary Targets Tracker

A small tool for GTA Online players prepping for The Kortz Center heist. It's a spreadsheet of all 65 secondary targets — paintings, gemstones, statues, jewelry, crates, and more — so you can see best seconadry target and how much it pays out.

The preview images also help you identify buyer requests, so you know exactly which item you're looking for.

Search by name, sort by any column, or filter down to a single category. Flip on hard mode to see payouts with the +10% bump, and peek at a preview image for anything you don't recognize by name alone.

**Status:** active — images being added as in-game captures become available.
